tools Package¶

Hermes: Tools

This package contains low-level tools for Hermes-specific datastructures and design patterns. Most notable is the Actor greenlet pattern, which is used in the DatastoreEngine component of the EventTracker.

actor Module¶

Tools: Actor Model

Allows a class to act as an Actor, which wakes up and processes a singular type of task once new tasks are added via the internal Gevent queue at Actor.inbox.

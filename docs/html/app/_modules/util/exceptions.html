# -*- coding: utf-8 -*-

'''

Hermes: Exceptions

Holds top-level app exception classes, including Hermes' package-level
exception ancestor, exported as `Error`.

-sam (<sam.gammon@ampush.com>)

'''


## Top-level Error Classes

[docs]class Error(Exception): ''' Top-level Hermes error. '''


[docs]class ClientError(Error): ''' Error that occurred because of a client-side slip-up. '''


[docs]class PlatformError(Error): ''' Platform (server-side)-related errors that occur in APIs or handlers. '''

## Tracker Errors


[docs]class TrackerError(PlatformError): ''' Error that occurred within the Tracker subsystem. '''


[docs]class EventError(ClientError): ''' Error that occurred because of improper event structure. '''


[docs]class InvalidSentinel(EventError): ''' Indicates that a URL came through in non-debug mode without a sentinel key. '''

## API Errors


[docs]class APIError(PlatformError): ''' Error that occurred within ProtoRPC-backed RPC subsystems. '''

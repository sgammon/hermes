raw(request, propagate=True)[source]¶

Entrypoint for recording raw events and producing model.raw.Event.

If requested, this method will also produce a guess as to how the request can be handled, if any. Available guess options are enumerated in a bidirectional protocol enum at protocol.flow.ResponseVector.

Parameters:
  • request – The current webapp2.Request, or a valid model.raw.Event to inject directly into the raw eventstream.
  • propagate – Flag indicating that this event should be propagated to the global eventstream. Defaults to True.
Returns:

Published (and possibly new) model.raw.Event, and, if requested via live, a guess about how to handle the request, like: tuple(<event>, <guess>).

subscribe(stream=None, _start=True)[source]¶

Subscribe to one-or-more event streams in EventTracker. Channels are persistently subscribed-to via Redis pubsub.

Iterables, strings, regexes (for PSUBSCRIBE) and None are acceptable as streams - None indicates a subscription to the global Event Stream.

Parameters:
  • stream – String channel name ("coolchannel"), regex channel name (r"coolch*") or iterable of those. Defaults to None, indicating a global EventTracker subscription is desired.
  • _start – Immediately schedule the subscription Greenlet for execution, before returning. Defaults to True.
Returns:

Result of the low-level subscribe operation, and the new Greenlet which will do the listening.

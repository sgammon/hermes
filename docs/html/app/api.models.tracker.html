class api.models.tracker.event.TrackedEvent(**properties)[source]¶

Bases: apptools.model.Model

A single hit to the EventTracker server, linked to a tracker.Tracker model via a string ID, and containing n parameters. Individual TrackedEvent models detail an individual tracking hit, with all associated information and metadata about aggregations and attributions.

Parameters:
  • raw – Reference apptools.model.Key to the raw.RawEvent that generated this particular TrackedEvent.
  • params – Python dict describing the interpreted parameters for this TrackedEvent. For raw parameters, see raw.RawEvent.
  • type – Describes the type of event being recorded in this TrackedEvent, such as a CLICK or IMPRESSION. These are enumerated in the bidirectional enum event.EventType.
  • tracker – Reference apptools.model.Key to the tracker.Tracker that recorded this TrackedEvent.
  • provider – Describes the provider that relayed this event to Ampush, if any. Available providers are enumerated in the bidirectional enum event.EventProvider.
  • aggregations – Lists aggregation groups that this TrackedEvent was found to be a part of. Aggregation groups listed in this area have already considered this TrackedEvent.
  • attributions – Lists attribution groups that this TrackedEvent was linked to. Aggregation groups listed in this area have already linked and considered this TrackedEvent.
  • modified – Timestamp datetime of the last time this TrackedEvent was modified.
  • created – Timestamp datetime of when this TrackedEvent was recorded.
